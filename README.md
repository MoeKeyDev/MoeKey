# MoeKey

MoeKey is a misskey client for Flutter

> This project is currently under development and has many functional deficiencies

## Features

- [x] misskey Servers Selection List
- [x] misskey Login
- [x] TimeLine
- [x] Drive
- [x] Notifications
- [x] Create,reply and forward Notes
- [x] Streaming Apis
- [x] Compatible with moekey themes file
- [x] Mfm text
- [x] Emoji reactions
- [x] Multi users login
- [ ] I18n
- [X] User Profile
- [ ] User Widgets
- [X] Clips
- [X] Explore
- [x] Announcements
- [x] Search
- [ ] User Lists
- [ ] Achievements
- [ ] Channels
- [ ] Firefish Apis
